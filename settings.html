API Key Settings

Enter your OpenAI API key to enable advanced prompt analysis.

How to Get Your OpenAI API Key

  1. Visit the OpenAI API Keys Page.
  2. Log in or sign up for an account if you haven't already.
  3. Click on "Create new secret key" and copy the generated key.
  4. Paste the key into the input field above and click "Save".

Your API key will be stored locally and used to enhance the prompt analysis features of this extension.
